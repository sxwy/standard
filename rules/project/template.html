脚手架

安装

  • node >= 16
npm i @sxwy/cli -g

使用

Usage: sxwy <command> [options]

Options:
  -V, --version          output the version number
  -d, --debug            是否开启调试模式 (default: false)
  -h, --help             display help for command

Commands:
  init [options] [name]  init project
  help [command]         display help for command

示例

  • 查看脚手架的使用方式
sxwy -h
  • 查看脚手架的版本
sxwy -V
  • 开启脚手架的 debug 模式
sxwy -d
  • 查看 init 具体的使用方式
sxwy init -h
  • 选择项目模板并自动下载
sxwy init
